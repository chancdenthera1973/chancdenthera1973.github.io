Album Artwork Assistant is designed to help you find and embed album artwork into iTunes tracks.

To use Album Artwork Assistant you must launch iTunes and select the track to which you want to embed metadata, then go to the application and press the Get Albums button.

Album Artwork Assistant will automatically select all the iTunes entries that share that particular album title. The resulting list contains all the tracks that will be modified.

The lower part of the window shows possible artwork matches found on the Internet on Google image search and amazon.com. If nothing useful shows up, change the search text and hit the Return key to try again, until you find a good match.
