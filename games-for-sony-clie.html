Wireless File Exchange for Classrooms
Bachmann Software has announced FilePoint Classroom Edition, a new workgroup software product aimed at solving the problem of wireless file exchange between teachers and students who use Palm OS handhelds in K-12 classrooms.

Using FilePoint Classroom Edition, teachers can create assignments, quizzes and other classroom materials on their workstations, and distribute them wirelessly to their classroom’s Palm OS handhelds over the network. Once on the handheld, students can work on their assignments, and individually submit them back to the teacher when completed.

FilePoint Classroom Edition works with any Palm OS handheld that is capable of achieving a wireless network connection, including PalmOne’s Tungsten C and T series, as well as any Palm OS handheld used in conjunction with the Enfora Wireless Portfolio or other Wi-Fi or Bluetooth wireless add-on. Sony Clie Landscape Patch Coming
tamspalm writes, recently, a company called MobileStream announced a landscape mode patch for the Clie series. Mobile Stream announces development of the utility which allows to use very popular Sony CLIE NX/NZ/TH series models in landscape mode. The utility implements standard PalmSource Pen Input Manager API 1.1. Practically this means that any application that works in landscape mode on Tungsten T 3, Tungsten T 5 or Zodiac handhelds will work in landscape mode on Sony CLIE handhelds.
