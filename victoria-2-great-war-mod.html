Issues fixed[edit]

Crashing on Windows 8/XP: map cache[edit]

This crash is caused by map cache data from an old version downloading with the game on Steam.
Fix crash on Windows 8/XP[citation needed]
  1. Go to <path-to-game>map
  2. Delete the cache folder.
  3. Go to %USERPROFILE%DocumentsParadox InteractiveVictoria 2map
  4. Delete the cache folder.
  5. Restart Victoria II and let it regenerate the map cache.

Crashing on Windows XP: Incompatible lua5.1.dll & lua51.dll files[edit]

This crash is caused by the included LUA scripting DLLs being incompatible with Windows XP, which prevent the game from starting.

Replace files[citation needed]
  1. Download older LUA DLLs version lua5_1_4_Win32_dll6_lib.zip.
  2. Go to the installation folder.
  3. Rename or remove the original lua5.1.dll and lua51.dll files.
  4. Extract lua5.1.dll and lua51.dll from lua5_1_4_Win32_dll6_lib.zip to the installation folder.

Notes

Alternatively, the lua5.1.dll and lua51.dll from Hearts of Iron III can be used.
Some versions of the game with the Heart of Darkness DLC installed may still crash in the middle of loading.

Historical Project Mod (HPM)[edit]

Requires Heart of Darkness DLC and beta 3.04
Adds significant historical flavor and features to the game, particularly through events and decisions
The Github wiki contains numerous minimods and addons
Install HPM
  1. Download the latest version of HPM.
  2. Extract HPM into the mod folder in the installation folder.
  3. Click the check box next to the mod's name in the game launcher

Historical Flavour Mod (HFM)[edit]

Fork of an older HPM version, adds a significant amount of nations & decisions but is considered more limiting in freedom as the events can railroad the player in historical scenarios
Install HFM
  1. Download the latest version of HFM.
  2. Extract HFM into the mod folder in the installation folder.
  3. Click the check box next to the mod's name in the game launcher
